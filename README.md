# Проектирование высоконагруженного аналога Яндекс Маркета
## 1. Тема и целевая аудитория
### 1.1 Описание темы
**Яндекс Маркет** - сервис компании Яндекс в сфере электронной коммерции. Состоит из маркетплейса с сетью постаматов, складов и пунктов выдачи заказов.

### 1.2 Целевая аудитория
**Количество посетителей в месяц:** 147.8 млн  
**Количество покупателей (пользователей, совершивших хотя бы одну покупку за год):** 18.2 млн  
**Количество запросов в поисковике Яндекс за декабрь 2025 года:** 3.39 млн  
**Онлайн-продажи, руб./мес.:** 535.3 млрд  
**Заказы, шт./мес.:** 126 млн  
  
**Распределение пользователей по возрастным группам:**

**Распределение по полу:**

**Распределение по странам:**  
Основной рынок находится в России - 95.56% трафика.


### 1.3 Функционал MVP
Для покупателя:
- Просмотр каталога товаров
- Поиск товаров по категориям, названию, модели и сравнение по параметрам и техническим характеристикам
- Корзина
- Оформление и оплата заказа
- Просмотр отзывов и возможность его оставить
- Участие в акциях (промокоды, Яндекс Плюс, скидки)

Для продавца:
- Инструменты аналитики и продвижения
- Инструменты логистики
  - Работа с ПВЗ (список доступных ПВЗ, поиск, рейтинг)
  - Выбор между FBY/FBS/DBS
