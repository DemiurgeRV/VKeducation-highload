## 2. Расчет нагрузки
### 2.1 Продуктовые метрики
Подтвержденной информации по месячной и дневной аудитории Яндекс Маркета нет. Из известной информации - количество покупателей  (пользователей, совершивших хотя бы одну покупку) за год составляет 18.2 млн человек [2], а средняя конверсия (процентное соотношение количества покупателей и посетителей) - 2% [6]. Исходя из этого рассчитаем примерное количество посетителей Яндекс Маркета:  
18.2 / 12 = 1.517 млн - среднее количество покупателей в месяц  
1.517 / 2 * 100 = 75.85 млн - среднее количество уникальных посетителей в месяц  
Поскольку информации о дневной аудитории также нет, используем коэффициент "липучести" (процентное соотношение DAU и MAU) максимально близкого аналога - WB. DAU wb - 14 млн, MAU wb - 92 млн [7].   
Значит коэффициент k = DAU wb / MAU wb * 100% = 14 / 92 * 100% = 15.22%  
Отсюда, DAU YM = 75.85 * 15.22% = 11.54 млн  
Общее количество заказов за год - 126 млн [2]. Отсюда, Среднее количество заказов на пользователя в год - 126 млн / 18.2 млн ≈ 7 шт
